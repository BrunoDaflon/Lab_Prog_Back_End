docker-compose
cria um file: docker-compose.yml

services:
  apache:
    image: httpd:latest
    ports:
      - "8080:80"

  nginx:
    image: nginx:latest           (o "image: nginx:latest" se usar "build: ."    ele procura um dockerfile) 
    ports:
      - "8081:80"    



Esses 2 serviços seria o mesmo se fizesse
docker run --name nginx -d -p 8081:80 nginx
docker run --name apache -d -p 8080:80 httpd

para rodar esse serviço:
docker-compose up -d
